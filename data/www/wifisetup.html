WiFi config

If SSID is set controller will attempt to connect to that Wifi network.
If SSID is empty controller will create a local network.
If AP SSID is set controller will use that name for a local network.
Minimum password length is %PASSLEN% characters.
Trailing spaces will be removed from text fields.

Other options are used for a static IP when connecting to a WiFi network and are not normally used.
Grey text shows the current DHCP values, these are not saved.

Controller MAC is %MAC%
