WiFi config

Admin password is used for this page and the wifi options reset page. To remove admin password enter REMOVE and submit.
If SSID to join is set controller will attempt to connect to that Wifi network.
If SSID to joim is empty controller will create a local network.
If AP SSID is set controller will use that name for a local network.
Minimum WiFi password length is %PASSLEN% characters.
Note all passwords are being transmitted over unencrypted HTTP.
Trailing spaces will be removed from text fields.

Other options are used for a static IP when connecting to a WiFi network and are not normally used.
Grey text shows the current DHCP values, these are not saved.

Controller MAC is %MAC%
